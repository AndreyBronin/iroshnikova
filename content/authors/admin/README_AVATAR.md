# Инструкция по замене аватара

Для установки нового фото Алены Ирошниковой как аватара:

1. Сохраните присланное изображение как `avatar.png` в эту папку:
   `/Users/bronin/my/iroshnikova/content/authors/admin/avatar.png`

2. Или используйте любой другой формат (jpg, webp), Hugo автоматически обработает изображение

3. Рекомендуемые параметры:
   - Размер: минимум 400x400 пикселей
   - Формат: PNG, JPG, WEBP  
   - Соотношение сторон: 1:1 (квадрат)
   - Качество: высокое для профессионального вида

4. После замены аватара пересоберите сайт:
   ```bash
   hugo server --disableFastRender
   ```

Текущее профессиональное черно-белое фото отлично подойдет для B2B сайта маркетолога.
